# 项目介绍

苍穹外卖（sky-take-out）是一款面向餐饮行业的外卖点餐系统，采用前后端分离架构，包含用户端和管理端两大核心模块。用户可通过微信小程序浏览菜品、下单支付、实时追踪订单状态；管理端提供员工管理、分类菜品管理、订单处理、数据统计报表等功能。项目完整覆盖了餐饮外卖业务的核心流程，是学习 Java 微服务开发的实践案例。

# 技术栈

- **Spring Boot 2.7.3**：应用开发框架
- **MyBatis + PageHelper**：数据持久层与分页组件
- **MySQL**：关系型数据库
- **Redis**：缓存
- **Druid**：数据库连接池
- **Knife4j**：API 接口文档
- **阿里云 OSS**：图片文件存储
- **WebSocket**：实时推送订单状态
- **Spring Cache**：方法级缓存
- **Apache POI**：数据报表 Excel 导出
- **Lombok / FastJson**：开发效率工具

# 功能设计

## 用户端

- **微信登录**：基于微信授权获取用户信息，JWT 生成 Token
- **浏览点餐**：按分类查看菜品与套餐，支持菜品口味选择
- **购物车**：添加/删除/清空购物车项
- **下单支付**：提交订单，接入微信支付回调
- **订单追踪**：查看订单状态，历史订单查询
- **地址管理**：管理收货地址

## 管理端

- **员工管理**：员工账号启用/停用、分页查询
- **分类管理**：菜品/套餐分类的增删改查
- **菜品管理**：菜品上下架、批量操作，含口味管理
- **套餐管理**：套餐组合、启售停售
- **订单管理**：订单搜索、接单、派送、完成、拒单
- **数据报表**：营业额统计、用户统计、订单统计、销量排行 Top10 及 Excel 导出
- **工作台**：今日数据概览、待办订单提醒

## 公共功能

- **图片上传**：阿里云 OSS 存储，支持菜品/套餐图片上传
- **营业状态**：管理端控制店铺营业/打烊开关
- **定时任务**：自动取消支付超时订单（15 分钟），自动完成配送超时订单（60 分钟）
- **WebSocket 推送**：订单状态变更实时推送至管理端
- **公共字段自动填充**：AOP 切面自动填充创建/更新时间与操作人
